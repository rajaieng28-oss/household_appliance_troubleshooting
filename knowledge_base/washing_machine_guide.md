# Washing Machine Troubleshooting Guide

## Problem: Machine Not Draining

### Possible Causes

* Blocked drain hose
* Clogged pump filter
* Faulty drain pump

### Troubleshooting Steps

1. Disconnect power.
2. Inspect the drain hose for blockages.
3. Clean the pump filter.
4. Test drain functionality.

### Recommended Solution

Remove hose obstructions and clean the filter.

### Safety Precautions

* Disconnect power before servicing.

---

## Problem: Excessive Vibration

### Possible Causes

* Uneven load distribution
* Unbalanced machine
* Worn shock absorbers

### Troubleshooting Steps

1. Redistribute clothing evenly.
2. Verify machine leveling.
3. Inspect suspension components.

### Recommended Solution

Level the machine and balance loads.

---

## Problem: Machine Not Spinning

### Possible Causes

* Broken drive belt
* Faulty lid switch
* Motor failure

### Troubleshooting Steps

1. Verify lid closure.
2. Inspect drive belt condition.
3. Test motor operation.

### Recommended Solution

Replace damaged belts or faulty switches.
