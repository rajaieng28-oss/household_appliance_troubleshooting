# Dishwasher Troubleshooting Guide

## Problem: Dishes Not Clean

### Possible Causes

* Clogged spray arms
* Dirty filter
* Improper loading

### Troubleshooting Steps

1. Clean spray arms.
2. Remove and wash filter.
3. Reload dishes correctly.

### Recommended Solution

Clean internal components regularly.

---

## Problem: Dishwasher Not Draining

### Possible Causes

* Blocked drain hose
* Clogged filter
* Faulty drain pump

### Troubleshooting Steps

1. Inspect drain hose.
2. Clean filter assembly.
3. Test drain pump.

### Recommended Solution

Clear obstructions and replace failed pump if necessary.

---

## Problem: Water Leakage

### Possible Causes

* Damaged door gasket
* Loose hose connections

### Troubleshooting Steps

1. Inspect gasket condition.
2. Tighten hose connections.

### Recommended Solution

Replace damaged seals.
